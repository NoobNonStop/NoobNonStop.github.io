My Skill-Set

JavaScript

JavaScript is used to add interactivity to the webpage.For example, you click a button and it responds by displaying a drop-down menu. This action, of the browser registering the click and the response is what JavaScript does.

Cascading Style Sheets (CSS3)

Cascading Style Sheets or CSS3 is used to style a page and its contents. For example, how big you want the heading to be, what color you want the background to be and what margins should there be between different elements of the webpage. In short, it's the decoration of the webpage.

HyperText Markup Language (HTML5)

HyperText Markup Language may be said to be the structure of a page. For example, what elements should be there in the page, how many headings and sections, how many images and links and so on. In a nutshell, it's the structure of the website.

Front-end Frameworks I Can Work With

Bootstrap 3

Bootstrap 3 is a front-end framework which makes writing code easier with properties categorized into classes and providing the users with color-coded classes.

Bootstrap 4

Bootstrap 4 is another front-end framework which is an upgraded version of Bootstrap 3. It is even more customizable since its layout is based on CSS-FlexBox and Grid.

JavaScript Libraries I Can Work With

jQuery

jQuery is a JavaScript library which shortens the code to be written significantly, and helps programmers to increase efficiency. jQuery is just a way of writing JavaScript code in a shortened way. To give an example, a program of 7 lines in JavaScript can be written in only one line in jQuery.
